Java Game Framework

In Progress

I'm currently working on making a flexible game framework in java that can be used to easily make simple java games. There are lots of robust frameworks already out there but they are so large and flexible it is hard for someone with limited experience to pick up and easily understand. It's something that is fun and hopefully other people will be able to use it when its done, although it still has a lot of work to be done on it. I love game "demakes" and I'm really looking forward to using this to demake all my old favorites.
